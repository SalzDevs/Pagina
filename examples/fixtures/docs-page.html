Loading pages

Pagina loads local files and remote URLs. Start from the documentation index or pass a path on the command line.

Basic usage

bun start examples/page.html
bun start https://example.com/guide.html

The loader resolves relative links against the current document location.

Glossary

Viewport
Terminal width and height used for layout and wrapping.
Display list
Paint commands emitted after layout: text spans and background fills.
