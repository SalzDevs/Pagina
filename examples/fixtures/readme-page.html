Pagina

A terminal browser for HTML pages. Install with bun install and run bun start.

Features

  • Keyboard-first navigation
  • Supported markup
    • Headings, paragraphs, links
    • Lists, tables, blockquotes
  • Local files and remote URLs

Commands

Key Action
Enter Follow focused link
u Go back
o Open URL prompt
